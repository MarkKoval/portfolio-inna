# Premium Portfolio (Next.js 15)

Ultra-modern portfolio starter with App Router, TypeScript, Tailwind, GSAP, Lenis smooth scroll, and validated contact form.

## Run locally

```bash
npm run dev
```

Then open `http://localhost:3000`.

## Stack

- Next.js 15 + React 19 + TypeScript
- Tailwind CSS
- GSAP animations
- Lenis smooth scroll
- React Hook Form + Zod validation

## Project structure

- `app/(routes)/page.tsx` — homepage sections
- `components/sections/*` — Hero, Projects, About, Contact
- `components/ui/*` — reusable UI primitives
- `lib/*` — hooks, animation helpers, utilities

## Deploy (Vercel)

1. Push repository to GitHub.
2. Import project in Vercel.
3. Keep default Next.js build settings.
4. Deploy.
